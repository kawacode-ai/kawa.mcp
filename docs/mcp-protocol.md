# MCP Protocol Specification

The MCP (Memory Coordination Protocol) defines the interface
between development environments and the Kawa Code 
intent memory system.

The protocol enables:

• recording developer and AI intent during development
• querying historical reasoning context
• aligning decisions across humans and AI agents

This repository contains the reference implementation
of the MCP protocol used by Kawa Code.

Independent implementations of the protocol are possible.
However, compatibility with the reference implementation
is recommended.
